
#  Password Reset App (MERN Stack)

A full-stack Password Reset Application built using **React, Node.js, Express, and MongoDB**.  
This app allows users to request a password reset link and securely update their password using a token-based system.

---

##  Features

- User Registration (API)
- Forgot Password (Generate Reset Token)
- Reset Password using Token
- Secure Token-based Authentication
- One-time Token Usage (expires after use)
- Responsive UI with Toast Notifications
- Clean and Modern UI Design

---

## 🛠 Tech Stack

### Frontend:
- React (Vite)
- Tailwind CSS
- Axios
- React Toastify
- React Router

### Backend:
- Node.js
- Express.js
- MongoDB (Mongoose)
